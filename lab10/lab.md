Lab 10
==============================

> Find the Hamming distance of X and Y where:
> (a) X = FE01 (hex notation)
>     Y = 7E13 (hex notation)
> (b) X = 0101 (binary notation)
>     Y = 1101 (binary notation)

## A

```
X = FE01 = 1111 1110 0000 0001
Y = 7E13 = 0101 1110 0001 0011
D        = X X          X   X
         = 4 / 16
         = 1 / 4
```

**Hamming distance: 1/4 = 0.25**

## B

```
X = 0101
Y = 1101
D = X
  = 1 / 4
```

**Hamming distance: 1/4 = 0.25**
